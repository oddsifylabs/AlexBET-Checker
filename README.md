# Telegram Bet Result Bot

Simple Telegram bot that checks NBA game results and tells you if your bet won.

## Usage

Send:
New York Knicks 4/30/2026

## Setup

1. Create bot via BotFather
2. Copy token

## Run locally

pip install -r requirements.txt
export TELEGRAM_BOT_TOKEN=your_token
python bot.py

## Deploy (Railway)

- Push to GitHub
- Connect repo on Railway
- Add env var:

TELEGRAM_BOT_TOKEN=your_token
